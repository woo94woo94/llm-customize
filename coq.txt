구분 : 전망
질문 : 당월 전망한 매출과 직접이익금액 실 단위로 알려줘
필요 데이터 : 당월 전망
PMS 메뉴 : 매출/실행원가 계획(M+3)

[질문 2]
구분 : 전망
질문 : M+3기준 매출 전망을 월별로 알려줘. 단,실행 중 프로젝트는 PM이 롤링한 최종 Rollong Data, 신규 프로젝트는 매출/실행원가 계획 메뉴에 반영되어 있는 수주 가능성 기준으로 전망해줘 (확실, 가능, 유동)
필요 데이터 : M+3 전망
PMS 메뉴 : 매출/실행원가 계획(M+3)

[질문 3]
구분 : 실적집계
질문 : 최근 2개년 IT사업실(조직명)의 매출실적을 PBS(사업분류)기준으로 집계해줘
필요 데이터 : 결산실적
PMS 메뉴 : 조직별 PJT손익실적(요약)

[질문 4]
구분 : 실적집계
질문 : 당해년도 영업이익이 적자로 수주된 프로젝트를 알려줘
필요 데이터 : 최초계약정보
PMS 메뉴 : PJT종합정보

[질문 5]
구분 : 실적집계
질문 : 최근 3개월 사업실별 실매출가동율을  알려줘 (※실매출가동율 : 매출프로젝트에 투입된 정규직 인력투입율)
필요 데이터 : 타임카드 등록현황
PMS 메뉴 : 노무비 계획/실적 현황

[질문 6]
구분 : 실적분석
질문 : 현재 실행 중 프로젝트 중 최종 실행원가가 최초계약대비 직접이익율이 5%이상 증가 및 감소한 프로젝트를 사업실별로 구분해서 보여줘
필요 데이터 : 최초계약정보, 최종롤링정보
PMS 메뉴 : PJT종합정보

[질문 7]
구분 : 실적분석
질문 : 전월 전망한 M+1월 매출 대비 당월 매출 실적의 gap이 큰 프로젝트 List를 알려줘 (gap: +/-1억원 이상)
필요 데이터 : 전망자료, 결산실적
PMS 메뉴 : 매출/실행원가 계획(M+3), 조직별 PJT손익실적(요약)

[질문 8]
구분 : 실적분석
질문 : 최근 2개년 최초계약이익율을 사업실별/PJT Type으로 분석하고 그래프로 표현하고 사유도 알려줘
필요 데이터 : 최초계약정보
PMS 메뉴 : PJT종합정보
