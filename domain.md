## 1. 프로젝트 기본 관리

### 프로젝트 마스터 정보
- `po_pjt_plan` (프로젝트 마스터)
- `pm_wbs_mst` (프로젝트 실행팀 및 매출액관리)

**설명**: 프로젝트의 기본 정보와 WBS(프로젝트 실행팀 및 매출액) 정보를 관리합니다. WBS는 Work Breakdown Structure가 아닌, 프로젝트 실행팀별 매출액을 관리하는 단위입니다.

### 프로젝트 Key Member
- `pm_key_mbr` (프로젝트 Key Member)

**설명**: 프로젝트에 투입되는 핵심 인력과 역할을 관리합니다.

### 프로젝트 마감 관리
- `pm_period_close` (프로젝트 마감 관리)
- `pm_period_late_apply` (결재중 실적보관 대상)

**설명**: 월별 프로젝트 마감 상태와 결재 중인 실적의 임시 보관을 관리합니다.

## 4. 원가 계획 관리

### 노무비 계획
- `pm_dclz_plan` (노무비 인력투입계획)
- `pm_tmplbr_lcst_plan` (임시직 노무비 계획)
- `pm_dclz_plan_hv` (MM Hold 실적포함 조회 VIEW)

**설명**: 프로젝트의 노무비 계획과 인력 투입 계획을 관리합니다. 정규직과 임시직을 별도로 관리합니다.

### 재료비/협력작업비 계획
- `pm_mcst_plan` (재료비/협력작업비 계획)

**설명**: 프로젝트의 재료비와 협력작업비 계획을 관리합니다.

### 경비 계획
- `pm_ecst_plan` (경비계획)

**설명**: 프로젝트의 경비 계획을 관리합니다.

### 공통비 계획
- `pm_comcst_ym_plan` (공통비 계획)

**설명**: 프로젝트의 공통비 계획을 연월별로 관리합니다.

## 7. 손익 관리

### 실행계획 손익
- `pm_pjt_exe_profit` (실행계획 손익종합)
- `pm_pjt_exe_profit_hold` (결재중인 경우 손익실적 임시보관)

**설명**: 프로젝트의 최초 계약 시점과 최종 계약 시점의 손익을 조회합니다.

## 도메인 간 주요 관계

### 프로젝트 중심 관계도
```
po_pjt_plan (프로젝트 마스터)
    ↓ 1:N
pm_wbs_mst (실행팀/매출액관리)
    ↓
    ├─ 계획 데이터 (pjt_cd + wbs_cd + ver)
    │   ├─ pm_dclz_plan (노무비 계획)
    │   ├─ pm_mcst_plan (재료비 계획)
    │   ├─ pm_ecst_plan (경비 계획)
    │   ├─ pm_comcst_ym_plan (공통비 계획)
    │   ├─ pm_tmplbr_lcst_plan (임시직 계획)
    │   └─ pm_revenue_plan (매출 계획)
    │
    ├─ 실적 데이터 (pjt_cd + wbs_cd)
    │   ├─ pm_lcst_prmpc_rsl (노무비 실적)
    │   ├─ pm_mcst_rsl (재료비 실적)
    │   ├─ pm_ecst_rsl (경비 실적)
    │   ├─ pm_comcst_othexp_rsl (공통비 실적)
    │   ├─ pm_revenue_cdit_rqst_rsl (매출 실적)
    │   └─ pm_timecard (타임카드)
    │
    └─ 손익 데이터 (pjt_cd + wbs_cd + ver)
        ├─ pm_pjt_exe_profit (실행손익)
        └─ pm_pjt_exe_profit_hold (임시손익)
```

### 참조 데이터 관계도
```
프로젝트/WBS
    ├─ cm_com_cd (공통코드) - 상태, 구분값
    ├─ cm_emp (직원정보) - 담당자
    ├─ cm_dept (부서정보) - 실행팀, 영업팀
    ├─ cm_costct_mst (코스트센터) - 원가센터
    ├─ po_cust_sites (고객사) - 발주처
    └─ cm_approval_trgter (결재정보) - 승인
```

## 주요 용어 정의

| 용어 | 설명 |
|------|------|
| **WBS** | Work Breakdown Structure가 아닌, 프로젝트 실행팀별 매출액 관리 단위 |
| **실행계획** | 프로젝트의 원가, 매출, 인력 계획 (버전으로 관리) |
| **실적** | 프로젝트의 실제 발생한 원가와 매출 |
| **손익** | 매출에서 원가를 차감한 이익 (계획/실적/최초/최종) |
| **노무비** | 인건비 (정규직 인력의 급여 관련 비용) |
| **재료비** | 프로젝트에서 사용하는 자재 및 물품 비용 |
| **협력작업비** | 외주 업체에 지급하는 비용 |
| **경비** | 여비교통비, 통신비 등 프로젝트 경비 |
| **공통비** | 프로젝트에 배부되는 간접비 |
| **인건비성경비** | 복리후생비, 퇴직급여 등 인건비 관련 경비 |
| **M/M** | Man Month (인력 투입량, 1인 1개월 기준) |
| **M/H** | Man Hour (인력 투입량, 1인 1시간 기준) |
| **Cost Center** | 원가를 관리하는 조직 단위 |
| **타임카드** | 직원의 일별 프로젝트 투입 시간 기록 |
| **BOQ** | Bill of Quantity (물량 내역서) |
| **결재** | 실행계획, 계약 등의 승인 프로세스 |
| **Hold** | 결재 중인 데이터의 임시 보관 상태 |
